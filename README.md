# Tesla Lock Sound - Minimal Checkout & Mailer

Tech stack: Node.js, Express, Stripe Checkout, Nodemailer, Bootstrap.

Run locally:

```bash
npm install
cp .env.example .env  # 手动创建并填写环境变量
npm start
```

Important:
- 请将 `.env`（Stripe、SMTP 等敏感信息）保存在本地，不要提交到 Git。
- 推送到 GitHub 前，确保未包含任何密钥或大文件（见 `.gitignore`）。
